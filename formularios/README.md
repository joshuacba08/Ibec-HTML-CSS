# FORMULARIOS

Tipos de formularios

```html
<form action="">
      <!-- input entrada de texto -->
      <input type="text" />

      <!-- input de tipo password -->
      <input type="password" />

      <!-- input de tipo email -->
      <input type="email" />

      <!-- input de tipo number -->
      <input type="number" />

      <!-- input de tipo date -->
      <input type="date" />

      <!-- input de tipo time -->
      <input type="time" />

      <!-- input de tipo color -->
      <input type="color" />

      <!-- input de tipo checkbox -->
      <input type="checkbox" />

      <!-- input de tipo radio -->
      <input type="radio" />

      <!-- input de tipo file -->
      <input type="file" />

      <!-- input de tipo submit -->
      <input type="submit" />

      <!-- input de tipo reset -->
      <input type="reset" />

      <!-- input de tipo button -->
      <input type="button" value="tipo button" />

      <select name="" id="">
        <option value="pizza">Pizza</option>
        <option value="empanada">Empanada</option>
        <option value="asado">Asado</option>
      </select>
    </form>
```
